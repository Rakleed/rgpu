Используем создаваемый в задании kodaktor.ru/g/modularity класс Таймер, 
     у которого метод start возвращает Promise
   Протестируем его работу с помощью mocha

   (этот фреймворк будем считать уже установленным)

   
	01. Создайте папку и войдите в неё в терминале
	02. yarn init

"scripts": {
	"test": "mocha --compilers js:babel-register"
  }

	03. yarn add --dev should babel-register babel-preset-env
----------------------------------------------------------------------------
	04. .babelrc
----------------------------------------------------------------------------
{
  "presets": ["env"]
}
----------------------------------------------------------------------------

	05. yarn add timerpromise или curl -kSLO 'https://kodaktor.ru/Timer.js'
----------------------------------------------------------------------------
	06. test.js
----------------------------------------------------------------------------
/*jshint esversion: 6 */
import Timer from 'timerpromise';  // или import Timer from './Timer';
import should from  'should';
const
      n=1,
      res = n*1000,
      timer1 = new Timer(n);

describe('#start', ()=> {
  it('return n*1000', ()=>
     timer1.start.should.eventually.be.exactly(res)
  );
});

	07. yarn test
		должны увидеть один пройденный тест


 Обратите внимание:  should.eventually автоматически работает как коллбэк в .then( ... )

	06. yarn add mongoose
	07. yarn test
		должны увидеть один пройденный тест
