Практическое задание 1. Текст задания¶

Формулировка¶

Напишите код на языке Python, позволяющий вычислить логические операции (задание 1), с помощью конкатенации строк, умножения строки на число или как-то иначе создайте таблицу и выведите её (задание 2) и, наконец, поработайте со списком чисел ряда Фибоначчи и кортежем (задание 3).

Разместите код на платформе repl.it (https://repl.it), ссылку на доску (или доски) разместите в форуме, решение задачи сопроводите соответствующими комментариями (при необходимости). Убедитесь, что в вашем решении есть ФИО автора и номер варианта. Для оформления используйте шаблон: https://repl.it/KuMT/2

Задание 1.¶

Логическая задача. Требуется для двух или трех входных переменных (A, B, C) построить таблицы истинности (см. свой вариант):

  1. Штрих Шеффера.
  2. Стрелка Пирса.
  3. A∧(B∨¬B∧¬C)
  4. (A∨B)∧(¬A∨¬B)
  5. (A∨B)∧ ¬C
  6. ((C∨B)→B)∧(A∧ B)→B
  7. (A→B)∧A) ↔ A∧B
  8. (A∧B ↔B∧C)∨(¬C →A)
  9. (A∨B ↔B∨C)∧(¬C↔A)
  10. A∨¬C→¬(B↔A)
  11. (A∧B ↔B∧C)∨(¬C →A)
  12. ¬A∨B→B→A∧B
  13. A∧¬B↔A∨B
  14. A∧C→B∨¬(C∧A)
  15. ¬B→C∧B∨(¬C→B)
  16. ¬(A∧B)∨A→B∨¬A
  17. ¬(A∨B)→¬(B∨C)
  18. A∨(B∧C)↔(A∨B)∧(A∨C)
  19. C∧B∨(¬B∨C→B∨C)↔C
  20. A∧B∨B→¬(A∧¬B↔¬A→B∧¬A)
  21. B∧¬C∨¬(¬B→C∧B↔B)
  22. ¬(¬(A∧B)∨A↔B)∨¬A
  23. ¬(A∨¬C∧A)↔¬(¬A∧C→C)
  24. ¬A∨B→(B→¬(A∧B))
  25. ¬(A∧B)∨(C∧B∧A)→¬(A∧¬C↔B∧C)∧¬A∧C∧B∨A

Инструменты для самопроверки:

  • http://www.wolframalpha.com
  • http://mathworld.wolfram.com/Equivalent.html

Задание 2.¶

Используя конкатенацию строк, функцию print(), а также умножение строки на число, создайте оформление для таблиц истинности, созданных в рамках предыдущего задания. Пример:

*******************************
*   A   *   B   *    A ⋁ B    *
*******************************
*  True * False *    True     *
*******************************

Дополнительным баллом будут отмечены те, кто реализуют это в виде, приближенном к современному виду таблиц («уголки, перекрестия тоже будут реализованы специальными символами).

Задание 3.¶

Для данного списка, используя слайсы, обращение к элементам по индексу (не используя циклы или условные операторы) найдите:

  1. Сумму четных элементов (со второго по предпоследний элемент) списка.

  8. Список элементов, стоящих на нечетных позициях в первой половине списка в обратном порядке.
  9. Последний элемент (цифры, из которых состоит элемент, должны идти в обратном порядке) из элементов второй половины списка.
  10. Отрицательное значение последнего элемента из элементов первой половины списка.

  12. Сумму всех элементов элементов списка с элементами, стоящими на четных местах в его первой половине.
  13. Сумму всех элементов списка с элементами, стоящими на нечетных местах в его первой половине.
  14. Сумму всех элементов списка с элементами, стоящими на четных позициях в его второй половине.
  15. Сумму всех элементов списка с элементами, стоящими на нечетных позициях в его второй половине.
  16. Сумму элементов, стоящих на четных местах списка с минимальным элементом этого списка.
  17. Сумму элементов, стоящих на нечетных местах списка с максимальным элементом этого списка.
  18. Список элементов, состоящий из элементов стоящих на четных позициях в первой половине списка и элементов , стоящих на нечетных позициях второй половины списка.
  19. Список элементов, состоящий из элементов стоящих на нечетных позициях во второй половине списка и элементов , стоящих на четных позициях первой половины списка.
  20. [для кортежа, см. ниже] Сумму всех чисел с плавающей точкой
  21. [для кортежа, см. ниже] Сумму всех целочисленных значений
  22. [для кортежа, см. ниже] Список названий полей (name, n_of_wheels, ...)
  23. [для кортежа, см. ниже] Список всех значений («DeLorean DMC-12», «rear», ... )
